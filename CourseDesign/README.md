# 单片机课程设计

## 题目

1. 循迹小车设计；扩展（尾随小车(引导车+尾随车)，遥控功能）
1. 避障小车设计；扩展(报警，语音功能)
1. 测速小车设计；扩展(速度超过限值报警，时间显示)

### 循迹小车

**功能：**
跟踪黑色条纹带。

**原理：**
红外传感器循迹，
当发光管发射的红外光遇到白色背景时，由于白色反射光强，反射大部分红外光至接收管，接收管收到大量红外光,输出为高电平；
而遇到黑色路径时，由于黑色吸光强，大部分红外光被吸收，以致接收管收到极少的红外光，输出为低。白色——高电平，黑色——低电平。

### 避障小车

**功能：**
能够对物体运动方向受到的阻碍作出各种躲避障碍的动作。

**原理：**
超声波模块：超声模块通过发射器向四面八方发射出去，发射出的超声波在空气中沿直线传播，遇到障碍物后发生反射反应。超声波接收器收到被障碍物反射回的超声波以后，经由内部计算得到障碍物的距离信号。

### 测速小车

**功能：**
实时测量小车的运行速度，并用液晶显示。

**原理：**
数字编码器：编码器发出的红外光，经过物体表面的反射并再次接收，智能车的轮子即使是黑色的，仍然足够反射红外线导致传感器的响应，当传感器“看到”车轮的一部分时，编码器会输出低电平，当它通过了车轮上的一个洞时，它的输出变为高电平。定时，然后计算脉冲个数，计算速度。

## 验收要求

1) 完成设计功能并当场演示设计成果、回答验收教师的提问、提交课程设计说明书(验收日提交)；
2) 对于附加功能，独立实现者与借鉴或抄袭别人成绩会有区别。

要求：课设可以两人一组，但报告每人一份，且不能雷同，如发现雷同，两人成绩均为零分。
